# 🧮 BMI Calculator

A simple and responsive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.  
This project helps users quickly calculate their BMI and understand their health category.

---

## 🚀 Features

- 📊 Calculate BMI using weight (kg) and height (cm)
- 🧠 Displays BMI result with category:
  - Underweight
  - Normal
  - Overweight
  - Obese
- 🎨 Clean and modern UI design
- 📱 Responsive layout (works on mobile & desktop)
- 📋 BMI categories table for reference
- ⚡ Instant calculation using JavaScript

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
